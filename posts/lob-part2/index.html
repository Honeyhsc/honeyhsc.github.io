Introduction

In this notebook, we will explore the L2 limit order book data and build a model to predict the best bid and ask prices. We will use the dataset provided to me during a college project, which contains the following columns:

  • timestamp: The time at which the order was placed.
  • bid: The price at which the order was placed.
  • ask: The lowest price at which someone is willing to sell.
